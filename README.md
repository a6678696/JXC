# JXC
SpringBoot进销存系统

本系统使用SpringBoot架构，数据访问层采用Spring Data Jpa，安全框架采用Shiro，实现了完整权限系统，Controller方法采用shiro注解，来实现有效的权限控制；界面采用了Easyui技术；
